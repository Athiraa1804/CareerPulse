# ShaktiDB Overview

## Introduction

ShaktiDB is an indigenous, PostgreSQL-compatible relational database system developed under the guidance of IIT Madras, IITM Pravartak Technologies Foundation, and C-DAC. It is designed to provide a secure, scalable, and high-performance database ecosystem suitable for enterprise, government, and research applications.

---

## Why ShaktiDB?

ShaktiDB aims to strengthen India's digital infrastructure by providing a sovereign database platform built on open-source PostgreSQL technology while extending it for large-scale and mission-critical workloads.

### Key Benefits

* PostgreSQL Compatible
* Open Source
* Enterprise Ready
* High Availability Support
* Distributed Database Architecture
* Advanced Security Features
* ACID Compliant Transactions
* Elastic Scalability
* Unified Management Console
* Compliance-Oriented Design

---

## Core Features

### PostgreSQL Compatibility

ShaktiDB is fully compatible with PostgreSQL drivers, tools, SQL standards, and extensions. Existing PostgreSQL applications can migrate with minimal changes.

### Security

ShaktiDB includes:

* SSL Encryption
* OAuth Integration
* Multi-Factor Authentication (MFA)
* Role-Based Access Control (RBAC)
* Session Revocation
* Audit Logging

### Distributed Architecture

ShaktiDB extends PostgreSQL with distributed database capabilities:

* Horizontal Scaling
* Coordinator–Worker Architecture
* Automatic Shard Rebalancing
* Multi-Level Partitioning
* Disaster Recovery Support

### High Availability

Provides:

* Replication
* Automated Backups
* Point-in-Time Recovery
* Failover Support
* Multi-Node Deployments

## ShaktiDB in CareerPulse

The CareerPulse Placement Management System uses ShaktiDB as its backend relational database.

Current tables include:

* students
* users
* skills
* student_skills
* interests
* companies

ShaktiDB manages:

* Student Profiles
* Authentication Data
* Skills and Interests
* Company Information
* Placement Eligibility Records
